Установка и запуск

1. Скачать себе репо и открыть его в IDE
2. В терминале (в папке проекте в командной строке или в терминале внутри IDE) прописать команду:
    pip install -r requirements.txt
3. В файле main.py строка 10 изменить пусть к драйверу
4. Запустить тесты используя команду: python -m pytest
